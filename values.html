rus_GoByExample: Типы данных

Go имеет различные типы данных, включая строки (string), целые числа (integer), числа с плавающей запятой (float), логические значения (boolean) и т.д. Вот несколько базовых примеров.


Строки, которые могут соединяться вместе при помощи знака +.

Целые числа и числа с плавающей запятой.


Логические выражения.
package main

import "fmt"

func main() {

   fmt.Println("go" + "lang")


   fmt.Println("1+1 =", 1+1)
   fmt.Println("7.0/3.0 =", 7.0/3.0)

   fmt.Println(true && false)
   fmt.Println(true || false)
   fmt.Println(!true)
}
