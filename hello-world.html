rus_GoByExample: Привет, мир

Наша первая программа напечатает классическое сообщение «Привет, мир». Вот полный исходный код.
package main
import "fmt"
func main() {
   fmt.Println("hello world")
}
Чтобы запустить программу, поместите код в файл
hello-world.go и используйте команду go run.

Иногда нам надо собрать наши программы в бинарные файлы. Мы можем сделать это, используя команду go build.


Затем мы можем выполнить собранный бинарный файл напрямую.
$ go run hello-world.go
hello world

$ go build hello-world.go
$ ls
hello-world hello-world.go

$ ./hello-world
hello world
